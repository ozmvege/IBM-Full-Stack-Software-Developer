# Software Architecture Questions and Answers

This document contains a set of multiple-choice questions (MCQs) related to software architecture concepts, along with their correct answers.

## Questions and Answers

### Question 1: Which of the following best describes software architecture?
**Answer:** It represents the earliest design decisions.

---

### Question 2: Which of the following are artifacts that result from designing the software architecture?
**Answer:** UML diagrams

---

### Question 3: Components in a well-structured design should be ___________________.
**Answer:** Tightly cohesive

---

### Question 4: Which of the following is an advantage of using UML diagrams when designing software architecture?
**Answer:** They serve to bring new team members up to speed quickly.

---

### Question 5: Which of the following best describes encapsulation?
**Answer:** Bundling data and methods to hide an internal state, so a componentâ€™s specific implementation is not exposed.

---

### Question 6: Which of the following best describes a service?
**Answer:** A unit of functionality that focuses on a solution to a business need, is deployed independently, and is reused by multiple systems.

---

### Question 7: Which statement is true regarding a 2-tier architecture?
**Answer:** The interface resides on client machines and makes requests to a server for data or services.

---

### Question 8: Which of the following architecture patterns are mutually exclusive?
**Answer:** Peer-to-peer and two-tier

---

### Question 9: Which of the following environments is intended for developers to use while they are actively coding the application?
**Answer:** Development

---

### Question 10: Which of the following is a security device that monitors traffic between an internal and an external network?
**Answer:** Firewall
